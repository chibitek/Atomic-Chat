Built privacy-first, not privacy-optional

Everything below ships in the open-source app. No telemetry, no lock-in.

Runs on-device

llama.cpp and Apple MLX models run fully offline. Your conversations never touch a server unless you choose a cloud model.

Switch models mid-thread

Start a draft on a tiny local model, finish it on GPT-5 or Claude. One conversation, any model, no copy-paste.

Real-time sync

Conversations stay live across macOS, iOS and Web. Send on your phone, watch it appear on your Mac instantly.

Agents with MCP tools

Give models web search, a code interpreter and filesystem access through the Model Context Protocol.

Encrypted API keys

Provider keys are encrypted with your device key before they ever sync. Atomic's servers only ever see ciphertext.

Open source, no lock-in

MIT-licensed and self-hostable. Export every conversation as Markdown or JSON whenever you want.

Local when it matters. Cloud when you need it.

You decide per message — Atomic never makes that call for you.
